# Environment Variables (Env)

এই সেকশনে আমরা শিখেছি কীভাবে সরাসরি পড ফাইলের ভেতর ভেরিয়েবল ডিফাইন করতে হয়।

### কেন ব্যবহার করবেন?
- ছোট এবং সাধারণ কনফিগারেশনের জন্য।
- যখন ডাটা অন্য কোনো পডের সাথে শেয়ার করার প্রয়োজন হয় না।

### প্র্যাকটিক্যাল কমান্ড:
```bash
# পড তৈরি করুন
kubectl apply -f env-pod.yaml

# ভেরিয়েবল চেক করুন
kubectl exec env-demo-pod -- printenv | grep APP
