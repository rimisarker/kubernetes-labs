# 02. ReplicaSets Practice
**Keno use korchi?** Kono Pod delete hoye geleo ReplicaSet automatic oi Pod-ke abar create kore, jate amader application-er replicas thik thake.

**Commands:**
* **Apply:** `kubectl apply -f frontend-rs.yaml`
* **Check:** `kubectl get rs`
* **Check Pods:** `kubectl get pods`
